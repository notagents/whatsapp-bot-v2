# **SECCION 5: FLUJOS POR CATEGORIA**

## **CAT 1: SEMILLAS**

### **Explicacion automaticas vs fotoperiodicas (usar cuando pregunten)**

"Las automaticas son mas rapidas, tienen un tiempo de vida de 3 meses aproximadamente, pero suelen tener menor produccion porque son plantas mas chicas y tienen la complicacion de que si se estresan detienen su crecimiento comenzando a florar sin importar su tamano.

Las fotoperiodicas tienen un tiempo de vida mas largo, pero nos permiten sacar mayor produccion y como nosotros manejamos los tiempos de flora si tenemos algun problema en el cultivo que la frene podemos darle tiempo a que recupere y siga creciendo.

Nosotros recomendamos mas las fotoperiodicas por estas razones."

### **Productos recomendados segun respuesta**

**Si elige FOTOPERIODICAS:**

- **Nacional recomendada:** Choco OG - blister x3, x5 y x10 semillas. Facil cultivo, bien productoras, flora corta de 8 semanas. Tonos oscuros en cogollos, aroma/sabor frutal-chocolate. Cuenta con garantia de germinacion. Link: https://www.astrogrow.com.ar/productos/choco-og-se/
- **Importada alternativa:** Amnesia de Buddha Seeds - blister x3 semillas. Clasico con predominancia sativa, productora, altos niveles de THC, sabor citrico. Garantia de germinacion. Link: https://www.astrogrow.com.ar/productos/amnesia-x3-buddha-seeds/

**Si elige AUTOMATICAS:**

- **Nacional recomendada:** Gucci Lemon Auto - sativas sabor limon, muy productoras, 80-90 dias totales, cruza de AK47. Blister x3 y x5 semillas. Garantia de germinacion. Link: https://www.astrogrow.com.ar/productos/gucci-lemon-auto-se

# **SECCION 6: PROTOCOLOS DE PLAGAS (DOSIFICACIONES)**

## **Prevencion / Infestacion minima**

**Productos:** Jabon potasico + aceite de neem

| Uso      | Dosificacion (por 1L agua tibia) |
| -------- | -------------------------------- |
| Prevenir | 1 tapa jabon + 3 ml neem         |
| Combatir | 2 tapas jabon + 3 ml neem        |

**Aplicacion:** Pulverizar toda la planta (arriba y abajo) hasta goteo. Mejor en manana o atardecer. Repetir 2x/semana hasta mitad floracion. Luego solo jabon hasta 15 dias antes de corte.

## **Infestacion moderada**

**Producto:** Phitonat Plus

| Uso         | Dosificacion (por 1L) |
| ----------- | --------------------- |
| Insecticida | 2 tapitas             |
| Preventivo  | 1 tapita              |

## **Infestacion severa (ej: aranita roja con telaranas)**

**Productos:** ABA + Oil 85

**Dosificacion floracion:** 14 gotas ABA + 1.5 tapas Oil 85 en 1L agua

**Seguimiento:** Luego aplicar Phitonat Plus como preventivo.

# **SECCION 7: GLOSARIO PARA PRINCIPIANTES**

| Termino            | Explicacion simple                                                 |
| ------------------ | ------------------------------------------------------------------ |
| Automatica / Auto  | Florece sola por tiempo, no por luz. Mas rapida (60-90 dias total) |
| Feminizada / Fem   | Semilla que da planta hembra (la que produce flores)               |
| Fotoperiodo / Foto | Depende de las horas de luz para florecer. Mas control, mas tiempo |
| Indoor             | Cultivo en interior (carpa, placard, habitacion)                   |
| Outdoor            | Cultivo en exterior (patio, balcon, campo)                         |
| Vegetativo / Vege  | Etapa de crecimiento antes de florar                               |
| Floracion / Flora  | Etapa donde la planta produce flores/cogollos                      |
| Sustrato           | La tierra/medio donde crecen las raices                            |
