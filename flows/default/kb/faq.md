# FAQ del bot

## ¿Qué hace este bot?
Responde mensajes de WhatsApp usando un asistente con memoria y, opcionalmente, documentación (KB).

## ¿Cómo activo/desactivo las respuestas automáticas?
Escribí "pausar bot" o "desactivar respuestas" para pausar. Para hablar con una persona, escribí "hablar con persona" o "operador".

## ¿Guardan mi información?
Se guarda contexto de la conversación (resumen y hechos) para dar respuestas coherentes. No se comparte con terceros.
